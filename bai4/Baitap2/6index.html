What is CSS

Cascading Style Sheets (CSS) is used to format the layout of a webpage.

With CSS, you can control the color, font, the size of text, the spacing between element, how element are positioned and laid out, what background images or background colors are to be used, different displays for different devices and screen sizes, and much more!
